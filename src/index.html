Help

This is a Web App and there is no "Backend". All data and files will stay on your device. The conversion is solely done by your computers resources.
Just click on the button to upload the files you want to convert, select the files and pick them up from your download directory.
Due to the nature of software development, and that I'm not related to Garmin in any way, it might happen, that future Xero Firmwares or Shotview versions will produce incompatible files. If you have created files with a newer version, or your file just doesn't work, just let me know on Github: Report an issue.

To download or not to download

This web app is downloadable, using "progressive web app" or "PWA" features. Due to it's nature to bridge from an app for ios and android to an app build on windows, that may or may not be cool for you. Unluckily, features differ from browser to browser and os to os. Here's a quick breakdown of features.

Offline Use

A PWA is basically a local copy of the site and and opens it in your browser when needed. It regularly checks for updates, when in use, so you don't need to uninstall and install manually.

OS integration - File association

Depending on the browser and os, this app can act as file share target. Once installed you can actually right-click a desired file, and the app will be listed in the "Open with ..." dialog. You can read more about the compatibility on MDN: PWA - file handlers.

OS integration - File Handler

Depending on the browser and os, this app can share files with other apps and not just store it into the download folder. You can read more about that on Web Share API.

Tested versions

The conversion was tested with the following versions
  • Garmin Xero Firmware: v1.37
  • Garmin Shotview for Android: v1.1.4
  • Garmin Shotview for iOS : v1.1.0

Convert a .fit file from the Garmin Xero C1 Pro device to labradar format

This uses the FIT SDK from Garmin(c). Just select the file, and you'll find it as csv in your download folder, with -xeroconv.csv at the end.

The files are located in the Garmin Shot_Sessions Folder on the Xero C1.

Convert a .csv file from the Garmin Shotview app to labradar format

This converts the csv file, you can export in Garmins Shotview App. Just select the file, and you'll find it as csv in your download folder, with a -xeroconv.csv at the end.

Convert a .xls / .xlsx file from the Garmin Shotview app to labradar format

This converts the xls(x) file, you can export in Garmins Shotview App. Just select the file, and you'll find it as separate csv in your download folder, with a -xeroconv.csv at the end.
